## 下载

### Docker

```
docker pull apachecn0/geeksforgeeks-ai-zh
docker run -tid -p <port>:80 apachecn0/geeksforgeeks-ai-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g geeksforgeeks-ai-zh
geeksforgeeks-ai-zh <port>
# 访问 http://localhost:{port} 查看文档
```
