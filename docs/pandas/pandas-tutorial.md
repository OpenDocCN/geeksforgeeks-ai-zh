# 熊猫教程

> 原文:[https://www.geeksforgeeks.org/pandas-tutorial/](https://www.geeksforgeeks.org/pandas-tutorial/)

**熊猫**是建立在 NumPy 库之上的开源库。它是一个 Python 包，提供了各种数据结构和操作来操作数字数据和时间序列。它主要是流行的，因为导入和分析数据容易得多。熊猫速度快，对用户来说具有高性能&生产力。
